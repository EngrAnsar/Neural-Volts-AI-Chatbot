# 🤖 Neural Volts AI Chatbot (No API Required)

A simple, intelligent customer support chatbot built using **Streamlit**, designed for businesses to automate responses without relying on paid APIs.

---

## 🚀 Overview

This project demonstrates a **rule-based AI chatbot** that simulates real customer support interactions. It is lightweight, easy to deploy, and ideal for:

- Customer support automation demos  
- Portfolio projects  
- Small business chatbot solutions  
- AI + Electrical/Automation service showcases  

---

## 🧠 Features

✅ No API required (100% free)  
✅ Built with Streamlit  
✅ Interactive chat interface  
✅ Predefined intelligent responses  
✅ Session-based chat history  
✅ Easy to customize for any business
